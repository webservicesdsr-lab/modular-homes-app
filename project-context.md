BLUEPRINT MAESTRO — MODULAR HOMES INTERACTIVE BUILDER
1. Qué estamos construyendo realmente

No estamos construyendo solo un website de modular homes.

Estamos construyendo una experiencia interactiva de venta, personalización y precalificación para casas modulares, enfocada especialmente en los acabados interiores, porque ahí está la mayor oportunidad comercial.

El producto debe sentirse como una mezcla entre:

una web premium
una app mobile-first
un configurador visual
una experiencia inspirada en House Flipper
una herramienta de ventas y lead generation

La idea central es esta:

el usuario entra por curiosidad, se engancha visualmente, empieza a “jugar” con opciones, entiende el valor de personalizar su casa, y eventualmente termina enviando un lead mucho más calificado.

2. Modelo de negocio detrás del producto

Este punto es clave y no debe perderse en futuros chats.

Negocio real

El jefe del usuario tiene un negocio de remodelaciones.

También existe una alianza con una persona que vende modular homes.

Cómo se gana dinero

Hay dos fuentes principales:

A. Comisión por venta de modular homes
Se generan leads para casas modulares
Si el lead compra la casa modular, se gana comisión
B. Ingreso por personalización / acabados / interiores
Después o durante el proceso, el cliente elige detalles interiores y exteriores
El equipo del usuario se encarga especialmente de la parte premium de acabados e interiores
Ahí está el margen más atractivo
Conclusión estratégica

La casa modular atrae al lead.
Los interiores y acabados generan el mayor valor.

Por eso el software debe vender la idea de:

“No solo compres una modular home. Personalízala para que se sienta tuya.”

12. Filosofía mobile-first

Esto es no negociable.

El usuario ha dejado claro que le importa más móvil que desktop.

Reglas móviles
títulos centrados
subtítulos centrados
descripciones centradas
jerarquía clara
botones grandes
spacing respirado
cards con lectura fácil
interacción táctil cómoda
sin depender de hover
Desktop

Puede tener más libertad compositiva, pero sin romper el sistema.

16. Resumen ejecutivo corto

Si necesitas una versión más corta para pegar arriba de todo en otro chat, usa esta:

Proyecto: Modular Homes Interactive Builder
Objetivo: Crear una experiencia tipo app / juego inspirada en House Flipper para vender modular homes y, sobre todo, impulsar la personalización de interiores, donde está el mayor margen comercial.
Negocio: comisión por venta de modular homes + ingreso por acabados/interiores premium realizados por el equipo del usuario.
UX/UI: SEO first, mobile-first, limpia, premium, animaciones suaves, sin ruido.
Estado actual: home page avanzado con hero fotográfico, CTA fuerte y carousel de cards centrado que muestra opciones de personalización como kitchen, bath, flooring, paint, lighting, etc.
